Introduction to AWT

Component

  • Component class is at the top of the hierarchy, it is an abstract class that encapsulates all of the attributes of a visual component
  • Except for menus everything else is subclass of component

Container

  • Subclass of component
  • has additional methods that allow other component objects like buttons etc. to be nested within it
  • Layout manager does this automatically
    • Flow
    • Grid
    • Border
    • Box
  • Subclasses of Container are Frame, Dialog and Panel
  • Frame is a subclass of window

Panel

  • Superclass of applet, i.e. a small program not intended to run on itâ€™s own but rather embedded inside another application

Frame

  • Frame() and Frame(String title) are used to create frames, a HeadlessException is thrown when we create a frame but the environment does not support user interaction
  • We have setSize(int width, int height) and setSize(Dimension newDim)
  • Window will not be visible until you call the setVisible() and you can set title using setTitle()
  • Origin of each window is the top left corner

Pixel

  • Picture Element - Represents any single point or dot on the screen

Example Code

import java.awt.*;
import java.awt.event.*;

public class AWT1 extends Frame {
    AWT1() {
        super("Title is Title");

        System.out.println("Starting the Frame");

        addWindowListener(new WindowAdapter() {
            public void windowClosing(WindowEvent e) {
                System.out.println("Closing the Frame");
                System.exit(0);
            }
        });
    }

    public void paint(Graphics g) {
        System.out.println("Printing");
        g.setColor(Color.blue); // sets blue color for drawing
        g.drawRect(40, 40, 200, 200);// display a rectangle to contain
        g.drawOval(90, 70, 80, 80);// to draw oval (face)
        g.drawOval(110, 95, 5, 5);// to draw a left eye
        g.drawOval(145, 95, 5, 5);// to draw a right eye
        g.drawLine(130, 95, 130, 115);// to draw a nose
        g.drawArc(113, 115, 35, 20, 0, -180);// to draw a mouth
    }

    public static void main(String[] args) {
        AWT1 f = new AWT1();
        f.setSize(800, 400);
        f.setVisible(true);
        f.repaint();
    }
}

        add(textArea);

        b.addActionListener(new ActionListener(){

        b2.addActionListener(new ActionListener(){

        add(b2);

            public void itemStateChanged(ItemEvent a){

            }
