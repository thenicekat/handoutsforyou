String

  • In Java, String is not a primitive type, nor is it an array of characters
  • String rather describes an object and cannot be edited once declared, but rather only new ones can be made
  • .length() gives the length of the string
  • We have concat, replace, substring and toLowerCase
  • toLowerCase() returns the same string, but every letter is in lower case
  • Strings are immutable, which means a string cannot change its properties but only a copy of it
  • Concat returns a new string with the string that's passed added to it
  • This function uses pass-by-value
  • Replace method replaces old char by new char, and this also returns a new string since strings are immutable
  • Substring gives the substring in the string but with the index starting from 0 and does not include the last number

StringBuffer

  • If a string can change, and will be accessed from multiple threads, use a StringBuffer because StringBuffer is synchronous so you have thread-safety
class StringBufferDemo 
{ 
	public static void main(String args[])
	{ 
		StringBuffer sb1=new StringBuffer("hello");
		StringBuffer sb3= new StringBuffer("TEMPLE");

		System.out.println(sb1); // hello
		System.out.println(sb1.capacity()); // 21

		System.out.println(sb1.length()); // 5

		StringBuffer sb2=sb1.insert(1,"tr");

		System.out.println(sb2); // htrello

		sb3=sb3.reverse();

		System.out.println(sb3); // ELPMET
	}
}

Regular Expressions

  • Pattern class is used to create the regex pattern
  • Matcher class is used to match the pattern with the string
    • boolean matches(): It returns true if the sequence and the pattern match. Otherwise, it returns false. It is to be noted that the entire sequence must match the pattern.
    • boolean find(): It can be used to determine if a subsequence of the input sequence matches the pattern. It returns true if there is a matching subsequence and false otherwise. It can be called repeatedly, allowing it find all matching subsequences. Each call to find() begins where the previous one left off
    • String group(): It can be used to obtain a string containing the last matching sequency by calling group(), i.e., it returns the matching string. If no match exists, IllegalStateException is thrown.
    • String group(int group): It returns the input subsequence captured by the given group during the previous match operation.
    • int groupCount(): Returns the number of capturing groups in this matcher's pattern.
    • int start(): It returns the start index of the previous match.
    • int end(): It returns the offset after the last character matched.
    • String replaceAll(String replacement): It replaces every subsequence of the input sequence that matches the pattern with the given replacement string.
    • String replaceFirst(String replacement): It replaces the first subsequence of the input sequence that matches the pattern with the given replacement string.
import java.util.regex.*;

public class Main
{
	public static void main (String[] args) {
	    String x = "Icecream Java1 IcecreamJava2";
	    Pattern pat = Pattern.compile("Java", Pattern.CASE_INSENSITIVE);
        Matcher mat = pat.matcher(x);
        while(mat.find()){
        	System.out.println(mat.group());
        }
        System.out.println(x);
        x = mat.replaceAll("C");
        System.out.println(x);
	}
}
