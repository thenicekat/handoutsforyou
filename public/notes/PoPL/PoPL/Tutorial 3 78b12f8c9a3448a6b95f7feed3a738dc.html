What is a storage class for variables in C? and what are the types of storage classes in C

  • Auto, Register, Extern, and Static
  • storage_class data_type variable_name
  • auto is the default. Can be only accessed within the block or function they have been declared and not outside them
  • When you declare a variable with static, it's memory is used in the heap
  • When you declare it inside a function it is available for other functions in the file
  • If its inside a file, outside some function, is it available for another file in the same package
  • Extern means that the variablr is declared somewhere else. It's same as global variable
  • Can get access to a variable that hasn't been declared yet
  • Static variables default value is 0
  • Register stores these values in registers neither in stack nor in heap but depends on the availability of the registers
